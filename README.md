# Chip&Dale Auto Telegram Mini App

Новый проект с нуля. Прошлый проект не используется и не изменяется.

## Что внутри

- `index.html` - точка входа мини-аппа.
- `styles.css` - вся визуальная часть.
- `app.js` - маршруты, каталог, фильтры, избранное и кнопки.
- `assets/` - изображения, подготовленные из референсов.

## Как запустить в VS Code

1. Откройте папку проекта в VS Code.
2. Запустите `index.html` через расширение Live Server или любой локальный статический сервер.
3. Для Telegram Mini App загрузите эту папку на HTTPS-хостинг и укажите ссылку в BotFather.

Мини-апп работает и вне Telegram, но если открыть его внутри Telegram, он автоматически вызовет `Telegram.WebApp.ready()` и `expand()`.

## Админка

Админка лежит отдельной страницей: `admin.html`.

Откройте ее на том же домене/Live Server, отредактируйте авто и список фото, нажмите `Зберегти`, затем перезагрузите `index.html`. Данные сохраняются в браузере через `localStorage`.


## Supabase version

Добавлены файлы:
- `supabase-config.js`
- `supabase-setup.sql`

Что сделать:
1. В Supabase открой SQL Editor.
2. Вставь содержимое `supabase-setup.sql`.
3. Нажми Run.
4. Загрузи файлы проекта на GitHub:
   `git add .`
   `git commit -m "Connect Supabase"`
   `git push`

После этого:
- витрина читает авто из Supabase;
- админка сохраняет авто в Supabase;
- если Supabase временно недоступен, приложение покажет локальные демо-данные.
